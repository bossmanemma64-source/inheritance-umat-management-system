 # UMaT Management System

 Overview
Python OOP case study demonstrating **inheritance, `super()`, and method overriding**. 
The system models a basic university structure with `Person` as the base class and `Student` + `Lecturer` as child classes.

 Case Study Requirements
1. Parent Class: `Person` with attributes `name`, `age` and method `display_info()`
2. Child Class 1: `Student` with `student_id` and `enroll_course()` method
3. Child Class 2: `Lecturer` with `employee_id` and `teach_course()` method  
4. Use `super()` to initialize parent attributes in child classes
5. Method Overriding: Override `display_info()` in one child class while calling parent method
